# Android OpenGL ES 学习计划

## 项目目标

构建一个**特效相机 App**，具备：
- 实时滤镜切换
- 滤镜参数调节
- 人脸贴纸
- 视频录制

## 学习周期

- **时长**：5 周（38 天）
- **每日**：约 1 小时
- **背景**：了解 OpenGL 概念 + 无实践 + 数学需复习 + Camera2 经验丰富

---

## 第 1 周：渲染管线与基础绘制

### Day 1：项目搭建 + 渲染管线
- **实践**：搭建项目，绘制纯色背景
- **原理**：GPU 渲染管线各阶段
  ```
  顶点数据 → 顶点着色器 → 图元装配 → 光栅化 → 片段着色器 → 帧缓冲
  ```

### Day 2：顶点着色器 + 第一个三角形
- **实践**：编写着色器，绘制三角形
- **原理**：归一化设备坐标 (NDC) [-1, 1]

### Day 3：片段着色器 + 颜色插值
- **实践**：渐变色三角形
- **原理**：光栅化、varying 插值

### Day 4：VBO + 索引缓冲
- **实践**：用索引缓冲绘制矩形
- **原理**：GPU 内存布局、GL_TRIANGLE_STRIP vs GL_TRIANGLES

### Day 5：正交投影 + 屏幕适配
- **实践**：绘制等比例图形
- **原理**：正交投影矩阵推导

### Day 6-7：复习
- 绘制彩色五角星

---

## 第 2 周：变换矩阵与纹理

### Day 8：向量与矩阵基础（数学复习）
- **原理**：向量点积、叉积、齐次坐标、矩阵乘法

### Day 9：平移矩阵
- **实践**：让三角形移动
- **原理**：平移矩阵推导

### Day 10：旋转矩阵
- **实践**：旋转动画
- **原理**：旋转矩阵推导（三角函数）

### Day 11：MVP 矩阵组合
- **实践**：组合变换
- **原理**：矩阵乘法顺序 P × V × M

### Day 12：纹理基础
- **实践**：给矩形贴图
- **原理**：UV 坐标映射、纹理坐标翻转

### Day 13-14：复习
- 纹理 + 变换综合动画

---

## 第 3 周：相机纹理与基础滤镜

### Day 15：SurfaceTexture + 外部纹理
- **实践**：相机预览到 GLSurfaceView
- **原理**：GL_TEXTURE_EXTERNAL_OES、OES_EGL_image_external 扩展

### Day 16：基础滤镜 - 亮度/对比度/饱和度
- **实践**：可调节的图像参数
- **原理**：片段着色器像素运算

### Day 17：基础滤镜 - 灰度/反色/Sepia
- **实践**：经典滤镜效果
- **原理**：RGB 颜色空间变换

### Day 18：基础滤镜 - 色调/色温
- **实践**：色彩调整滤镜
- **原理**：HSV/HSL 颜色空间

### Day 19：滤镜架构设计
- **实践**：可切换滤镜的管理器
- **原理**：滤镜抽象与复用

### Day 20-21：复习
- 滤镜面板 UI + 参数调节

---

## 第 4 周：高级滤镜与特效

### Day 22：卷积滤镜 - 模糊
- **实践**：高斯模糊、均值模糊
- **原理**：卷积核与图像处理

### Day 23：卷积滤镜 - 边缘检测/锐化
- **实践**：Sobel 边缘检测、锐化效果
- **原理**：梯度与边缘

### Day 24：FBO + 多 pass 渲染
- **实践**：滤镜叠加（模糊+调色）
- **原理**：离屏渲染管线

### Day 25：LUT 滤镜
- **实践**：3D LUT 专业滤镜
- **原理**：颜色查找表

### Day 26：美颜滤镜
- **实践**：磨皮效果
- **原理**：双边滤波思想

### Day 27-28：复习
- 高级滤镜组合

---

## 第 5 周：贴纸特效 + 综合项目

### Day 29：多纹理混合
- **实践**：叠加 Logo/贴纸
- **原理**：混合模式 (Blend Mode)

### Day 30：人脸检测集成
- **实践**：ML Kit 人脸检测
- **原理**：人脸关键点获取

### Day 31：动态贴纸定位
- **实践**：贴纸跟随人脸
- **原理**：坐标转换（人脸坐标 → 纹理坐标）

### Day 32：动态贴纸动画
- **实践**：眨眼/张嘴触发动画
- **原理**：帧动画 + 状态机

### Day 33：3D 贴纸基础
- **实践**：简单的 3D 贴纸效果
- **原理**：透视投影 + 贴纸

### Day 34：录制与保存
- **实践**：录制带特效视频
- **原理**：MediaCodec + OpenGL

### Day 35-38：综合项目开发
- 特效相机 App 完整实现
